# Caroline's Portfolio

A collection of data analysis projects using Excel, SQL and Power BI focused on business insights and dashboard reporting.

## Excel Dashboard Projects  
**Tools:** Microsoft Excel, Power Query, Pivot Tables, Charts, Slicers


### Project 1: Firm Size Analysis Dashboard (Excel)
View Dashboard Screenshot [https://github.com/c-mutuku/Data_Analysis_Portfolio/blob/main/Screenshots/Firm_Size.png]

This project is an Excel dashboard analyzing firm size distribution, employment, and business dynamics across different firm size categories.

#### How to Use
1. Download and open the Excel file.
2. Go to the **Firm Size Dashboard** sheet.
3. Use the slicers to filter by firm size and year.
